Search for a file

Search for files

query Parameters
path
string

Search for a path that contains this value

path_re
string <regex>

Search for a path using Regex

tag_whitelist
string

Whitelist for exact tags that must exist on returned files.

tag_blacklist
string

Blacklist for exact tags that must cant exist on returned files.

count
integer [ 1 .. 200 ]
Default: 50

Number of files to return, max 200

index
integer
Default: 0

Index in the file list to start at

sort
string
Default: "none"
Enum: "none" "size" "stars" "date" "id" "random"

How should the files be sorted. Ascending by default. None using SQL default sorting (By ID), random shouldn't be used with index, as the random order changes every time.

sort_reverse
boolean

Sort descending instead of ascending

Responses
